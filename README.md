## Install

```shell
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX:PATH=/usr ..
make
```

## Uninstall

```shell
rm /usr/share/backgrounds/cutefishos/default.jpg
rm /usr/share/backgrounds/cutefishos/wallpaper-1.jpg
rm /usr/share/backgrounds/cutefishos/wallpaper-2.jpg
```
